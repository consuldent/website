# 🚀 GetHyped Clone – Animated Agency Website

A high-fidelity clone of the GetHyped agency website, built to practice modern UI animation, smooth interactions, and real-world frontend architecture.

---

## 🔥 About the Project

This project recreates a premium agency-style landing page with a strong focus on motion design, smooth scrolling, and interactive UI elements.

The goal is to simulate how real-world modern websites are built using animation libraries like GSAP combined with a scalable frontend structure.

---


## ✨ Features

- 🎯 Smooth scroll-based animations
- ⚡ GSAP-powered UI interactions
- 🎨 Clean and modern design system
- 🧩 Reusable component architecture
- 📱 Fully responsive layout
- 🧠 Performance-focused animation handling

---

## 🛠 Tech Stack

- **Next.js (App Router)**
- **Tailwind CSS**
- **GSAP (GreenSock Animation Platform)**
- **Lucide React Icons**
